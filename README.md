## Projeto em destaque

### Churn Prediction ML Platform

Projeto de Machine Learning voltado à previsão de cancelamento de clientes (churn), com foco em antecipar riscos e permitir ações preventivas de retenção.

O desenvolvimento do modelo de ML encontra-se em andamento, com foco na construção de um pipeline estruturado e evolutivo, preparado para cenários reais de negócio.

A solução está sendo construída considerando não apenas o modelo preditivo, mas todo o ciclo necessário para aplicação em produção:

- Pipeline de dados preparado para evolução contínua  
- Estrutura de preprocessing (cleaning, encoding e feature engineering)  
- Modelos preditivos (baseline e rede neural) para comparação de desempenho  
- Avaliação de trade-offs entre falsos positivos e falsos negativos  
- Estrutura de API para inferência e integração com sistemas  
- Base para monitoramento, versionamento e operação do modelo  

O projeto simula um cenário real de negócio, onde a previsão é utilizada como suporte à tomada de decisão, priorização de clientes em risco e redução de perda de receita em operações B2B2C.

## Próximos passos

Como evolução deste projeto, será desenvolvida uma camada adicional baseada em agentes de IA com LLMs, com o objetivo de transformar as previsões do modelo em ações práticas.

Essa camada permitirá:

- Análise contextual das previsões de churn  
- Geração de recomendações de retenção  
- Execução automatizada de ações preventivas  
- Integração com fluxos operacionais e sistemas externos  

A proposta é evoluir de um modelo preditivo para uma solução completa de decisão e ação orientada por IA.
