## Projeto em destaque

### Churn Prediction API

Projeto de Machine Learning voltado à previsão de cancelamento de clientes (churn), com foco em antecipar riscos e permitir ações preventivas de retenção.

A solução foi estruturada considerando não apenas o modelo preditivo, mas todo o ciclo necessário para aplicação em ambiente real:

- Pipeline de dados preparado para evolução contínua  
- Modelos preditivos (baseline e rede neural) para comparação de desempenho  
- Avaliação de trade-offs entre falsos positivos e falsos negativos  
- API de inferência para integração com sistemas externos  
- Estrutura base para monitoramento e operação em produção  

O projeto simula um cenário real de negócio, onde a previsão é utilizada como suporte à tomada de decisão, priorização de clientes em risco e redução de perda de receita.
